Intro-to-puredata
=================

Pure Data y Max/MSP son entornos de programación visual utilizados 
en proyectos interactivos y de multimedia en tiempo real. Estos 
entornos proveen una aproximación intuitiva a la manipulación de 
contenido y data en los nuevos medios.
Pd (Pure Data) es libre, gratuito y multi-plataforma, corriendo 
en Windows, Linux, Mac OS X y BSD.   Con la librería libpd, se 
puede utilizar a pd como un "audio engine" para el desarrollo de 
video juegos y aplicaciones para Android, iOS, y Raspberry Pi.
